# Massive Unity Integration

> [!WARNING]  
> UNSTABLE, DO NOT USE IN PRODUCTION! Stable version will come soon.

This package contains [Massive ECS](https://github.com/nilpunch/massive-ecs) integration for Unity.

Planned features:
- Entity & component setup in Unity scenes.
- Client-side visualization with standard Unity MonoBehaviours.
- Automatic view synchronization.
- Entity view preview.
- Runtime editing.
- Multi-edit support.
